# vim-simpleline
simply custom the vim statuline

Installation:
1. For vim
   a. Create the same path as the repo in your ~/.vim.
   b. Put vimscripts into these dirs.
2. For neovim
   a. Create the same path as the repo in your ~/.config/nvim.
   b. Put vimscripts into these dirs.
